### ☑️ 모바일게임 백야극광을 모티브로 한 턴제 전략 게임입니다.
- 게임 루프를 적용하는 프레임워크를 구성했습니다.
- 트리 자료구조와 그래프 탐색 알고리즘을 사용해 던전 스테이지를 생성했습니다.
- BFS와 A* 알고리즘을 활용한 길찾기로 플레이어와 적군의 전투를 구현했습니다.
- 상태패턴, 전략패턴, 컴포지트 패턴 등의 디자인 패턴들을 적용하여 개발했습니다.
- 본격적인 기획부터 개발까지 담당한 프로젝트로 게임이 만들어지고 돌아가는 과정을 이해할 수 있었습니다.
